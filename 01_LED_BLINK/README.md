# LED Blink using Arduino

## Description

This project demonstrates how to blink an external LED using an Arduino Uno. It is one of the basic Arduino projects and helps in understanding digital output, pin configuration, and delays.

## Components Used

- Arduino Uno
- Red LED
- 1kΩ Resistor
- Jumper Wires

## Connections

- Arduino Pin 8 → Resistor → LED (+)
- LED (-) → GND

## Working

The Arduino turns the LED ON for 1 second and then OFF for 1 second. This process repeats continuously, making the LED blink.

## Output

The LED blinks continuously with a delay of 1 second between ON and OFF states.
