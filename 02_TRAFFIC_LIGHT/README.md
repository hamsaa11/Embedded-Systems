# Traffic Light using Arduino

## Description

This project is a simple traffic light system built using an Arduino Uno. Three LEDs are used to represent the red, yellow, and green traffic lights. The LEDs turn ON one after another with different delays to simulate the working of a real traffic signal.

## Components Used

- Arduino Uno
- Red LED
- Yellow LED
- Green LED
- 3 - 220Ω Resistors
- Jumper Wires

## Connections

- Red LED → Pin 13
- Yellow LED → Pin 12
- Green LED → Pin 11
- Negative terminal of all LEDs → GND

## Working

When the program starts, the red LED glows for 3 seconds. After that, the yellow LED glows for 1 second, followed by the green LED for 3 seconds. This sequence repeats continuously, just like a traffic signal.

## Output

The LEDs glow in the order:
- Red
- Yellow
- Green

and this cycle repeats continuously.
